# Airflow on Kubernetes

Welcome to the technical documentation for the **Airflow on Kubernetes** project. This project is designed to help you set up a robust, scalable local development environment for Apache Airflow.

## 🚀 Quick Links

- **[Project Architecture](/docs/architecture)**: Understand the components and how they fit together.
- **[Lessons Learned & Findings](/docs/findings)**: Deep dive into the technical choices and best practices.
- **[GitHub Pages Setup](/docs/setup/github-pages)**: Learn how this documentation is deployed.

## 🛠 Project Goal

The primary goal of this repository is to demonstrate how to run Airflow using the **KubernetesExecutor** on a local **KinD** cluster, providing a production-like experience on your local machine.
